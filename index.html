psd.js

A Photoshop PSD file parser in Javascript for browsers and NodeJS

About

psd.js is a general purpose file parser for PSD files created in Photoshop. Given a PSD file, it can parse out information such as image size and color modes, image resources, layer info, image contents, etc.

The psd.js source files are written in Coffeescript and hosted on Github.

View Source View Documentation npm Info

Goals

psd.js is highly versatile. Some things it can do are:

  • Output flattened image to PNG
  • Output image to HTML canvas
  • Extract image contents of single layers
  • Parse important image data such as layer size, position, etc.

Example Usage

An extremely simple example of how psd.js can be used to access PSD layer information.

{PSD} = require 'psd'

psd = PSD.fromFile './path/to/file.psd'
psd.parse()

for layer in psd.layers
  console.log "Layer: #{layer.name}"
  console.log "Size: width=#{layer.cols}, height=#{layer.rows}"
  console.log "Position: top=#{layer.top}, left=#{layer.left}"

psd.toFile "/path/to/output.png"
